AutoComplete自动补全插件

AutoComplete运用

Ajax自动补全插件仅依赖jQuery，它能轻松的为文本框创建自动下拉补全功能，其压缩过后文件大小只有2.7K。

Html：

  1. <input type="text" id="autocomplete">

Ajax 请求：

  1. $('#autocomplete').autocomplete({
  2. serviceUrl: '/path/your.json',
  3. dataType: 'json',
  4. onSelect: function (suggestion) {
  5. console.log('游戏：' + suggestion.value + ', 原折扣：' + suggestion.fill + ', 现折扣：' + suggestion.refill);
  6. }
  7. });

本地数据：

  1. var gameData = [
  2. {value: "天天飞车", fill: "4.5", refill: "4.6"},
  3. {value: "天天跑酷", fill: "4.1", refill: "5.0"},
  4. {value: "天龙八部", fill: "3.7", refill: "4.3"},
  5. {value: "守护者们", fill: "4.5", refill: "5.2"},
  6. {value: "热血街霸3D(送春丽)", fill: "4.5", refill: "5.2"},
  7. {value: "画江湖之灵主（暴力美学指尖感触）", fill: "8.5", refill: "7.2"},
  8. {value: "神雕侠侣", fill: "7.5", refill: "5.6"}
  9. ];
  11. $('#autocomplete').autocomplete({
  12. lookup: gameData,
  13. onSelect: function (suggestion) {
  14. console.log('游戏：' + suggestion.value + ', 原折扣：' + suggestion.fill + ', 现折扣：' + suggestion.refill);
  15. }
  16. });

全部参数

参数 说明
serviceUrl
服务器请求地址		 类型：字符串，函数，默认值: null
返回服务器端URL或回调函数。 如果提供了本地查找数据(lookup)，则可选
lookup
查找数组或对象		 类型：数组，默认值: null
suggestion 组成的数组，suggestion 可以是字符串数组或对象数组.
suggestion: 单体数据字符串或后面这种 { 'value': 'string', 'data': 'any'}
lookupFilter
过滤函数		 类型：函数
数据为本地时对数据进行过滤的函数，默认只会进行局部匹配
onSelect
选择后的回调函数		 类型：函数，默认值: null
选择suggestions之后的回调函数，this对象和绑定的input关联
minChars
最小字符数		 类型：数字，默认值: 1
maxHeight
最大高度		 类型：数字，默认值: 300
suggestions容器的最大高度（以像素为单位）
deferRequestBy
请求间隔时间		 类型：数字，默认值: 0
keyUp之后发起请求的间隔时间
width
宽度		 类型：数字，字符串，默认值: 'auto'
suggestions容器的宽度（以像素为单位）
params
 类型：对象
与请求一起传递的其他参数，可选
formatResult
格式化函数		 类型：函数，默认值: 默认格式化函数
格式化suggestions容器中的suggestion entry, 可选
delimiter
分隔符		 类型：字符串或者正则，默认值: null
splits字符串并把得出数组的最后一项作为query进行请求.当需要分别对两个或者以上的词汇进行自动完成时很有用
type
类型		 默认值: GET
suggestions的Ajax请求类型
noCache
是否缓存		 类型：布尔值，默认值: false
设置是否缓存ajax结果的布尔值
onSearchStart
开始查询		 类型：函数，默认值: noop
Ajax请求之前调用；可用于前端验证，this 被绑定在input上
onSearchComplete
查询完成		 类型：函数，默认值: noop
请求成功之后的回调，this 被绑定在input上
onSearchError
 类型：函数，默认值: noop
如果ajax请求失败，则调用，this 被绑定在input上
tabDisabled 类型：布尔值，默认值: false
设置为true的话，用户选择结果之后，鼠标指针会blur，默认行为是focus
transformResult 类型：函数
Ajax请求完成后用于调节response.suggestions的格式.
appendTo
追加到...		 默认值: document.body
suggestions 容器被插入的地方
zIndex
容器层级		 类型：数字，默认值: 9999
