ScrollSpy插件 （滚动监听，集成在bootstrap中，经测试可单独拿出来使用）

  • 使用说明
  • 用法
  • 须知

使用说明

需要Bootstrap导航

Scrollspy目前需要使用Bootstrap导航组件来正确突出显示活动链接。

需要相对定位

scrollspy都需要使用position：relative;相对定位来监视元素。 在大多数情况下，作用在<body>上，当然也可以放在其它元素上。

用法

  1. 通过data属性

    根据情况，给需要监视的页面元素添加 data-spy="scroll" (一般是 body 元素，本例是在 <div class="scrollSpy-body fn-mb-20"></div> )上，并且通过 data-target 属性指定目标

    1. body{
    2. position: relative;
    3. }
    1. <body data-spy="scroll" data-target=".navbar-example">
    2. ...
    3. <div class="navbar-example">
    4. <ul class="nav nav-tabs" role="tablist">
    5. ...
    6. </ul>
    7. </div>
    8. ...
    9. </body>

  2. 通过 Javascript 语句

    当你在CSS中添加了position：relative;后，可以通过Javascript来监听滚动

    1. $('body').scrollspy({
    2. target: '.navbar-example'
    3. });

须知

导航代码中 ul 含有一个 CSS 类 .nav，它是必需的，没有的话就会导致插件无效果

Bootstrap 文档推荐第一种方法，因为不用任何的 JavaScript 语句即可实现 ScrollSpy 效果。

使用总结（可用以下两种方式分别在本页测试）

  1. 通过data属性来实现

    向要监听的元素（通常是 body）添加 data-spy="scroll"，本例要监听的内容是 .scrollSpy-body，监听的菜单是 #scroll_spy，必须要用 div包起来，不然会没有效果，代码如下：

    1. <div class="scrollSpy-wrap" id="scroll_spy">
    2. <ul class="nav navbar-nav scrollSpy-nav">
    3. <li><a href="#spy_1"></a>菜单一</li>
    4. ...
    5. </ul>
    6. <div class="scrollSpy-body fn-mb-20" data-spy="scroll" data-target="#scroll_spy">
    7. ...
    8. <div>
    9. </div>

  2. 通过Js来实现

    1. //Div代码
    2. <div class="scrollSpy-body fn-mb-20">
    3. ...
    4. <div>
    5. //Js代码
    6. $('.scrollSpy-body').scrollspy({
    7. target: '#scroll_spy'
    8. })
  3. 另一个通过Js来实现的例子是作用在 body上的，可查看 Affix粘附导航
