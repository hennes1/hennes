使用说明：

  1. 请确定已经引入了 smartMenu.css 和 smartMenu.js

  2. 定义右键菜单数据 （查看本例代码）

    1. var rightMenu = {
    2. text: "功能按钮1", //菜单名称
    3. icoClass: 'glyphicons glyphicons-magic', //菜单前的图标样式名，多个用空格隔开；如果没有，此参数可省略
    4. func: function () { //菜单执行函数
    5. var id = Hennes.getCurrentEleID(this); //获取当前元素id，可自定义
    6. layer.alert('你在ID为' + id + '的行中点击了'); //弹窗插件
    7. }
    8. },
    9. rightMenu2 = { //可定义多个右键菜单
    10. ...
    11. };

  3. 装载自定义菜单数据

    1. var loadMenuData = [
    2. [rightMenu, rightMenu2, ...]
    3. ];

  4. 执行自定义函数

    1. Hennes.rightMenu(e, loadMenuData);

    Hennes.rightMenu：为自定义封装的右键函数
    e：执行右键菜单的元素
    详情可查看本示例源码和common.js文件
